# 游戏创新发散与收敛流程（通用）

## 1. 先定创新北极星
- 用一句话定义：这次创新想提升什么体验或结果。
- 约束必须同时写明：时间、技术、预算、团队能力。

## 2. 发散维度
- 核心循环：操作、反馈、目标达成方式。
- 成长系统：能力成长、资源循环、构筑选择。
- 对抗关系：PVE/PVP、环境挑战、协作竞争。
- 内容结构：关卡、模式、事件、地图机制。
- 元玩法：局外养成、收集、赛季、社交。

## 3. 发散产出要求
- 至少 8 个候选创意。
- 至少包含 2 个保守增强、2 个中度融合、2 个激进探索。
- 每个创意必须写“核心乐趣变化点”。

## 4. 收敛评分模型（1-5）
- 新颖性：是否形成明显差异化记忆点。
- 可实现性：现阶段团队是否可按期做出。
- 乐趣潜力：是否明显提升可玩性与复玩动力。
- MVP 适配度：是否适合放进第一阶段验证。
- 风险可控性：失败成本是否可接受。

建议总分：
`总分 = 新颖性*0.25 + 可实现性*0.25 + 乐趣潜力*0.25 + MVP适配度*0.15 + 风险可控性*0.10`

## 5. 决策输出格式
- 稳定基线方案：确保可交付。
- 创新主方案：优先验证的差异化方向。
- 备选方案：主方案风险超阈值时切换。

## 6. 创新实验最小闭环
- 假设：玩家会因 X 机制而产生 Y 行为变化。
- 最小实验：用最小开发量验证假设。
- 指标：至少 1 个行为指标 + 1 个主观反馈指标。
- 终止条件：若连续 N 次未达阈值则暂停/淘汰。
